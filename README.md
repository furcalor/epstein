Epstein PDF Archive (Originals and JPGs extracted from PDFs)

This repository contains publicly released documents from the U.S. Department of Justice related to the Epstein case.

The original files are available at the DOJ website:
https://www.justice.gov/epstein/

To make browsing easier, each PDF has had JPG images extracted so the pages can be viewed quickly without needing a PDF reader.

Contents:
 -Original DOJ‑released PDF files (where included)
 -Corresponding JPG image sets for easier viewing
 -Video files
 -Other files included in original ZIP packages

Purpose:
This repo exists solely to improve accessibility and readability of the publicly available documents.
No modifications have been made to the content beyond format conversion and/or extraction.

Source:
All documents originate from the U.S. Department of Justice’s public release page:
https://www.justice.gov/epstein/
