Epstein PDF Archive (Converted to JPG)
This repository contains a selection of publicly released documents from the U.S. Department of Justice related to the Epstein case.
The original files are available at the DOJ website:
https://www.justice.gov/epstein/
To make browsing easier, each PDF has been converted into JPG images so the pages can be viewed quickly without needing a PDF reader.
Contents
• 	Original DOJ‑released PDF files (where included)
• 	Corresponding JPG image sets for easier viewing
• 	Simple batch tooling used for PDF → JPG conversion
Purpose
This repo exists solely to improve accessibility and readability of the publicly available documents.
No modifications have been made to the content beyond format conversion.
Source
All documents originate from the U.S. Department of Justice’s public release page:
https://www.justice.gov/epstein/
